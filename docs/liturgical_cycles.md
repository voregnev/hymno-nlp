# Богослужебные круги

## Три независимых цикла

Конкретная служба = пересечение всех трёх кругов + правила Типикона.

### 1. Суточный круг (фиксирован)

| Служба | `office` | Источники текстов |
|---|---|---|
| Малая вечерня | `vespers_small` | Октоих |
| Великая вечерня | `vespers_great` | Октоих + Минея |
| Повечерие | `compline` | Часослов |
| Полунощница | `midnight` | Часослов + Октоих |
| Утреня | `matins` | Октоих + Минея |
| 1-й час | `hours_1` | Часослов |
| 3-й час | `hours_3` | Часослов |
| 6-й час | `hours_6` | Часослов |
| 9-й час | `hours_9` | Часослов |
| Литургия | `liturgy` | Октоих + Минея |

### 2. Седмичный круг (Октоих, 8 недель × 7 дней = 56 уникальных дней)

| `day_of_week` | День | Тема |
|---|---|---|
| 0 | Воскресенье (Неделя) | Воскресение Христово |
| 1 | Понедельник | Небесные силы бесплотные |
| 2 | Вторник | Иоанн Предтеча |
| 3 | Среда | Крест, Богородица |
| 4 | Четверг | Апостолы, Николай Чудотворец |
| 5 | Пятница | Крест, Богородица |
| 6 | Суббота | Все святые, усопшие |

### 3. Годичный круг

#### Неподвижный (Минея)
Привязан к календарной дате, не зависит от Пасхи.
```python
menaion_month = 1   # январь
menaion_day   = 15  # 15 января
```

#### Подвижный (Триодь + Пятидесятница)
Зависит от даты Пасхи. Кодируется как смещение от Пасхи:

| `pascha_offset` | Период |
|---|---|
| -70 | Начало Триоди (Неделя о мытаре и фарисее) |
| -63 | Неделя о блудном сыне |
| -56 | Мясопустная неделя |
| -49 | Неделя сыропустная |
| -42 .. -8 | Великий пост |
| -7 | Страстная седмица |
| 0 | Пасха |
| +1 .. +6 | Светлая седмица |
| +7 | Антипасха (Фомина неделя) |
| +14 .. +48 | Пятидесятница |
| +49 | День Пятидесятницы |
| +56 | Неделя Всех Святых |

## Разделы богослужения (section)

### Вечерня и утреня (общие)

| `section` | Название | Греческий термин |
|---|---|---|
| `lihc` | На Господи воззвах | Εἰς τό, Κύριε ἐκέκραξα |
| `aposticha` | Стихиры на стиховне | Εἰς τὰ Ἀπόστιχα |
| `litya` | Лития | Λιτή |
| `praises` | Хвалитные стихиры | Εἰς τοὺς Αἴνους |
| `kathisma` | Седальны | Κάθισμα |
| `hypakoe` | Ипакои | Ὑπακοή |
| `exapost` | Светилен/Экзапостиларий | Φωταγωγικόν / Ἐξαποστειλάριον |

### Канон (утреня)

| `section` | `ode_number` | `unit_type` |
|---|---|---|
| `canon` | 1–9 (кроме 2*) | `hirmos` |
| `canon` | 1–9 | `troparion` |
| `canon` | 1–9 | `theotokion` |
| `canon` | выбранные | `katavasia` |

*Ода 2 поётся только в Великий пост.

### Типы единиц в каноне

| `unit_type` | Описание |
|---|---|
| `hirmos` | Ирмос — тематическая песнь, задающая мелодию оды |
| `troparion` | Тропарь — основная строфа оды |
| `theotokion` | Богородичен — заключительный тропарь в честь Богородицы |
| `katavasia` | Катавасия — повторение ирмоса в конце оды |

## Как Типикон определяет службу

Типикон задаёт правила совмещения источников. Например, воскресная служба 8-го гласа складывается так:

```
Вечерня (Великая):
  lihc:      10 стихир — 4 из Октоиха глас 8, воскресные
                        + 3 из Октоиха глас 8, анатолиевы  
                        + 3 из Минеи
  aposticha: 4 стихиры из Октоиха глас 8
  тропарь:   воскресный тропарь глас 8

Утреня:
  canon:     Октоих глас 8 (воскресный канон)
           + Октоих глас 8 (крестовоскресный или богородичный)
           + Минея
```

Реализация правил Типикона — **отдельный модуль** (Alpha+), не входит в MVP.
В MVP хранятся только тексты как они расположены в книгах,
без вычисления конкретного расписания на день.
