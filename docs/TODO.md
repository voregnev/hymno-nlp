### Морфологический анализ — лемматизация

Без лемматизации невозможно связать словоформу «душу́» с леммой «душа́»
и открыть нужную словарную статью.

- [ ] **Stanza pipeline для ЦСЯ** — модель `cu` из UD OCS-PROIEL
  ```python
  import stanza
  nlp = stanza.Pipeline('cu', processors='tokenize,lemma,pos')
  doc = nlp("душу")
  lemma = doc.sentences[0].words[0].lemma  # → "душа"
  ```
- [ ] **Таблица `word_forms`** в PostgreSQL:
  ```sql
  CREATE TABLE word_forms (
      id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
      unit_id UUID REFERENCES text_units(id),
      surface TEXT NOT NULL,   -- словоформа как в тексте
      lemma   TEXT NOT NULL,   -- нормализованная лемма
      pos     TEXT,            -- часть речи
      position_in_unit INT,
      lang    CHAR(3)
  );
  CREATE INDEX ON word_forms (lemma, lang);
  CREATE INDEX ON word_forms (surface, lang);
  ```
- [ ] **Скрипт `src/nlp/lemmatize.py`** — прогон всего корпуса через Stanza,

- [ ] **Endpoint `GET /wordform/{surface}?lang=cu`** — вернуть лемму + все
